Usage

These lexicons define the schema for records stored in the ATProto network. To use them in your application, reference them by their full URL:

https://[your-username].github.io/at_polis/com/crown-shy/testing/poll.json

About ATProto Polis

ATProto Polis is a decentralized deliberation platform built on the AT Protocol, inspired by Polis. It enables distributed, user-controlled polling and statement voting.
