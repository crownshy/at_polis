# AT Polis 

This repo contains a proof of concept for running a polis conversation on ATProto 

## Privacy warning 

Any statements or votes made on these polis polls are public and identifiable with 
the user who posted them 

## This is super experimental and so might not be working 

Hopefully it will though! 

## Starting development 

Running the api server is as simple as 

```bash
cargo run
```

Running the frontend is 

```bash
cd frontend
pnpm install
pnpm dev
```
